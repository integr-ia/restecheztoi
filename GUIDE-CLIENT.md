# Guide d'utilisation — Espace de gestion du site

Bienvenue ! Ce guide vous explique comment **modifier vous-même les menus et les photos** de votre site, sans aucune connaissance technique.

Vous pouvez tout faire depuis :
- 💻 Votre **ordinateur** (recommandé pour les longues modifications)
- 📱 Votre **téléphone** (parfait pour changer une photo rapidement)

---

## 1. Se connecter à votre espace

1. Ouvrez votre navigateur (Chrome, Safari, Edge…)
2. Allez sur l'adresse que votre développeur vous a communiquée :

   👉 **`https://restecheztoi.sanity.studio`**

3. Cliquez sur **« Continue with email »** (ou avec Google si vous préférez)
4. Entrez l'email avec lequel vous avez été invité
5. Vous y êtes !

> 💡 **Astuce** : ajoutez la page à vos favoris (étoile en haut du navigateur) pour y revenir facilement. Sur smartphone, vous pouvez aussi « Ajouter à l'écran d'accueil » pour avoir une icône directe.

---

## 2. La page d'accueil de l'admin

À gauche, vous voyez le menu **« Contenus du site »**, avec une seule rubrique : **Menus**.

Cliquez dessus → vous voyez vos 3 menus :
- Menu Découverte
- Menu Classique
- Menu Haut de Gamme

Cliquez sur celui que vous voulez modifier.

---

## 3. Modifier un menu existant

Quand vous ouvrez un menu, vous voyez tous ses champs. Voici à quoi ils servent :

| Champ | À quoi ça sert |
|---|---|
| **Titre du menu** | Le grand titre affiché sur le site (ex : « Menu Découverte ») |
| **Sous-titre** | Le petit texte sous le titre (ex : « L'art de bien commencer ») |
| **Prix (CHF)** | Juste le nombre (ex : `130`). Pas besoin d'écrire « CHF ». |
| **Format du menu** | Ex : « 3 plats » ou « 5 services » |
| **Image de bannière** | La grande photo en haut de la page du menu |
| **Image vignette** | La petite photo qu'on voit sur la page « Nos menus » |
| **Propositions de menus** | Les variantes du menu (Hiver, Végétarien, etc.) — voir plus bas |
| **Texte de bas de page** | Le petit message en bas de la page (optionnel) |

### Pour modifier un texte
Cliquez simplement dans le champ → tapez votre nouveau texte → cliquez sur **« Publish »** en bas à droite. **C'est tout.** Le site se met à jour en quelques secondes.

> ⚠️ Tant que vous n'avez pas cliqué sur **Publish**, vos modifications sont **enregistrées en brouillon** mais **pas encore visibles sur le site**. Pratique : vous pouvez préparer vos changements à tête reposée et publier seulement quand tout est prêt.

---

## 4. Changer une photo

1. Ouvrez le menu concerné
2. Trouvez le champ **« Image de bannière »** ou **« Image vignette »**
3. **Glissez votre photo** depuis votre ordinateur vers la zone de l'image
   - Sur smartphone : touchez la zone → « Upload » → « Take a photo » ou « Photo Library »
4. Sanity vous propose un cadre orange qu'on appelle le **« hotspot »** : c'est le point de l'image qui sera toujours visible, même si l'image est rognée pour s'adapter à différents écrans. **Faites glisser ce cercle sur le visage / l'élément important** de la photo.
5. Cliquez sur **« Publish »**.

### Conseils pour les photos
- **Bannière** : format paysage, large (au moins 1600 pixels de large)
- **Vignette** : format carré ou paysage, plus petite c'est ok (au moins 800 pixels)
- Préférez des photos **lumineuses** et de **bonne qualité**
- Pas de photos avec des textes en surimpression (ils seront difficiles à lire sur mobile)

---

## 5. Modifier les plats d'un menu

Faites défiler jusqu'à la section **« Propositions de menus »**. Chaque proposition (par exemple « Découverte Hiver ») contient une liste de plats.

### Pour modifier un plat existant
1. Cliquez sur la proposition (ex : « Découverte Hiver »)
2. Cliquez sur le plat à modifier (ex : « Entrée — Tatin de betteraves »)
3. Modifiez le **nom du service** (« Entrée », « Plat », « Dessert »…) ou la **description** (« Tatin de betteraves »)
4. Cliquez en dehors pour fermer, puis sur **« Publish »** en bas

### Pour ajouter un nouveau plat
1. Dans la proposition → tout en bas de la liste des plats → **« Add item »**
2. Remplir le **Nom du service** et la **Description**
3. **Publish**

### Pour supprimer un plat
1. Cliquez sur les **« ⋮ »** (3 petits points) à droite du plat
2. **« Remove »**
3. **Publish**

### Pour réorganiser les plats
**Maintenez** le plat avec votre doigt/souris et faites-le glisser vers le haut ou le bas.

---

## 6. Ajouter une nouvelle proposition de menu

Vous voulez par exemple créer un menu « Découverte Été » à côté du « Découverte Hiver » ?

1. Dans le menu → section **« Propositions de menus »** → **« Add item »**
2. Remplir le **titre** (« Découverte Été »)
3. Cocher **« Menu végétarien »** si c'est le cas
4. Ajouter les plats un par un avec **« Add item »**
5. **Publish**

---

## 7. Une erreur ? Pas de panique

À côté du bouton **Publish**, il y a un menu **« ⋮ »** :
- **« Discard changes »** : annule toutes vos modifications non publiées et revient à la dernière version publique.
- **Onglet « History »** (icône horloge en haut à droite) : montre toutes les versions précédentes — vous pouvez restaurer n'importe quelle version d'il y a 1 heure, 1 jour, 1 mois… 🕰️

**Conclusion : vous ne pouvez rien casser de définitif.**

---

## 8. Voir le résultat sur le site

Après avoir cliqué sur **Publish** :
- Attendez **30 secondes à 1 minute**
- Allez sur votre site `https://restecheztoi.com` (ou rafraîchissez la page si elle est déjà ouverte avec `Ctrl+F5` / `Cmd+Shift+R`)
- Vos modifications sont là ! 🎉

## En résumé

| Vous voulez… | Vous faites… |
|---|---|
| Changer un texte | Cliquez dans le champ, tapez, **Publish** |
| Changer une photo | Glissez la nouvelle image, ajustez le cercle orange, **Publish** |
| Ajouter un plat | Section « Propositions » → plat → **« Add item »** |
| Annuler une bêtise | **« ⋮ »** → **« Discard changes »** |
| Restaurer une version d'avant | Icône horloge 🕰️ en haut à droite |

---

## Vous avez un blocage ?

Contactez votre développeur — l'historique de toutes les modifications est sauvegardé, donc rien n'est jamais définitivement perdu.
